# LogDB Samples

Sample applications for the public LogDB .NET packages:
- `LogDB.Sample.SDK`
- `LogDB.Sample.OpenTelemetry`
- `LogDB.Sample.Serilog`
- `LogDB.Sample.NLog`

Documentation:
- https://docs.logdb.dev/

NuGet source:

```txt
https://api.nuget.org/v3/index.json
```

Quick start:

```bash
dotnet restore
dotnet build LogDB.Sample.sln
```
